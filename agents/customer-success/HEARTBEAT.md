# HEARTBEAT.md — Customer Success Manager

Run this checklist on every heartbeat (scheduled: Wednesday 9am CET).

## 1. Health Scan
Review all active trials and customers:
- Green (active <7 days): all good, no action
- Yellow (7–14 days inactive): send check-in message today
- Red (>14 days inactive): escalate to CEO, send personal email within 24h

## 2. Early Adopter Program Status
For each enrolled EA:
- Where are they in the journey? (Signup / Onboarding call / Month 1 call / Month 3 call / Converted)
- Any overdue milestones? Schedule the missing touchpoint.
- Any feature requests from EA calls this week? → Synthesize for Head of Product

## 3. Pending Onboarding Calls
- Any new EA signups waiting for their onboarding call? → Book within 48h
- Onboarding call prep: check if they have data to import (Excel or Winner Flex)

## 4. New Trial Activations
- Any new trials that signed up in the last 3 days?
- Day 0: Confirm welcome email sent
- Day 3: Check if they've created their first project. If not → activation nudge

## 5. Feedback Synthesis (Friday)
Compile weekly feedback report for Head of Product and CEO:
- Customer pain points this week (ranked by frequency)
- Feature requests (with customer name and verbatim quote)
- Bug reports (flag critical ones to CTO immediately)
- NPS comments received

## 6. Case Study Pipeline
- Any customer active ≥1 month with clear measurable results?
- If yes → schedule 30-min case study interview
- Draft in progress? → review and send to Growth Marketer for publication
