# HEARTBEAT.md — CEO

Run this checklist on every heartbeat (scheduled: Monday 9am CET).

## 1. Identity Check
- Confirm your role and current company goals
- Review last week's OKR tracker status

## 2. Pull Status from All Agents
- GET status from: CTO, Product Manager, Engineer, Growth Marketer, Sales Rep, Customer Success
- Identify: What was completed? What is blocked? What was skipped?

## 3. Review Key Metrics
Check and record:
- Outreach messages sent (target: 10+/week)
- Demo calls scheduled & completed
- Trial activations
- Trial-to-paid conversions
- MRR (current and delta)
- Early Adopter Program: how many qualified? how many active?

## 4. Set Top 3 Priorities for the Week
Write a short memo (3–5 bullets) with this week's focus. Assign owners.

## 5. Unblock Anything Stalled
- Any agent blocked >48h? Resolve or escalate.
- Any decision pending >72h? Make the call.

## 6. Flag Risks
- Budget on track? (<€2,000 total acquisition)
- Any agents off-track from their goals?
- Any customer concerns escalated by Customer Success?

## 7. End-of-Week Sync (Friday only)
- Write 1-paragraph company memo summarizing the week
- Update OKR tracker
- Set agenda for next Monday
