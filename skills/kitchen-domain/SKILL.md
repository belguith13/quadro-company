## The Kitchen Sales Workflow (6 Stages)

Quadro CRM's pipeline maps exactly to how kitchen businesses operate:

1. **Discovery (Prospection)** — First customer contact; needs assessment, showroom visit
2. **Quote (Devis)** — Line-item quote created, versioned, sent as PDF via email
3. **Sale (Vente)** — Customer accepts quote; project becomes a confirmed sale
4. **Order (Commande)** — Components ordered from supplier(s)
5. **Delivered (Livré)** — Components at depot or customer site; delivery doc generated
6. **Closed (Terminé)** — Installation complete; project archived

## Key Pain Points in Kitchen Businesses

- **Quote versioning chaos**: Multiple quote versions sent by email, customer doesn't know which is current
- **Order tracking in Excel**: No visibility on what's ordered, what's arrived, what's missing
- **Installer scheduling conflicts**: Two installations booked same day because scheduling is on a whiteboard
- **No delivery documentation**: Delivery docs created manually in Word, inconsistent format
- **Fragmented tools**: CRM in one app, quotes in Excel, calendar in Google, orders tracked by phone call
- **Winner Flex gap**: Winner Flex handles 3D design and ordering but has no CRM, scheduling, or delivery workflow
